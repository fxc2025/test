文件分类说明
================================================================================

UI 设计文件
-----------
这些是高保真的 HTML 页面，可以直接在浏览器中打开查看。
包含首页、仪表盘和预览页，完全响应式设计。

业务逻辑分析
-----------
针对 m2_users 自动创建时机和用户名重复检查的详细分析。
包含问题分析、解决方案、实施指南和快速参考。

数据库设计
----------
Supabase PostgreSQL 数据库脚本。
improved 版本修复了业务逻辑问题，添加了辅助函数。

产品文档
--------
完整的产品需求文档、API 设计和项目说明。
v3 版本是最新的优化版本。
