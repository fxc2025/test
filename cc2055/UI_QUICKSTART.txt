🚀 快速查看
================================================================================

方式一：直接打开（最简单）
------------------------
1. 找到 cc2055/ui-preview.html
2. 双击在浏览器中打开
3. 点击页面上的按钮查看首页和仪表盘

方式二：使用本地服务器（推荐）
----------------------------
cd cc2055/

# Python 3
python -m http.server 8000

# Node.js
npx http-server .

# 然后在浏览器访问
http://localhost:8000/ui-preview.html

方式三：VS Code Live Server
--------------------------
1. 安装 Live Server 扩展
2. 右键 ui-preview.html
3. 选择 "Open with Live Server"

🎨 设计特点
================================================================================

色彩系统
--------
主色调: #3498db (蓝色)
成功色: #2ecc71 (绿色)
警告色: #f39c12 (橙色)
危险色: #e74c3c (红色)
渐变色: linear-gradient(135deg, #667eea, #764ba2)

交互效果
--------
✅ 卡片悬停抬起
✅ 按钮悬停变化
✅ 平滑滚动
✅ 淡入动画
✅ 导航栏滚动效果

响应式断点
----------
移动端: < 640px   (单列布局)
平板:   641-1024px (两列布局)
桌面端: > 1024px   (完整布局)

🔧 技术实现
================================================================================

纯 HTML + CSS + JavaScript
--------------------------
✅ 无需构建工具
✅ 无需依赖库
✅ 可直接在浏览器打开
✅ 易于理解和修改

使用的现代特性
--------------
✅ CSS Grid - 响应式布局
✅ Flexbox - 元素对齐
✅ CSS Variables - 主题颜色
✅ Intersection Observer - 懒加载动画
✅ Media Queries - 响应式设计

💡 使用建议
================================================================================

1. 开发参考
   - 查看 HTML 结构作为组件设计参考

2. 改造建议
